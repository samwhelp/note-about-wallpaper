
## Remove

``` sh
make remove
```

## Usage

### On File Manager

1. Launch FileManager. For example: `thunar`
2. Specific Image File
3. Open With `opset-wallpaper-lxqt` to set wallpaper

### On lxqt4-terminal

1. Launch `lxqt4-terminal`
2. Run `opset-wallpaper-lxqt image_file`
